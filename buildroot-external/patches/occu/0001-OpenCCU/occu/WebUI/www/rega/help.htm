Offline-Hilfe
Bitte legen Sie die beiliegende HomeMatic-CD in ihr CD- bzw. DVD-Laufwerk ein. Die CD startet automatisch. Sollte die Hilfe nicht innerhalb von 15 Sekunden starten, ist Ihr Laufwerk nicht auf einen solchen automatischen Start eingestellt. Bitte gehen Sie dann unter "Start => "Ausführen" auf "Durchsuchen" Stellen Sie den Laufwerkbuchstaben für das CD-Laufwerk ein und wählen Sie die Datei Setup.exe aus. Folgen Sie den Anweisungen am Bildschirm.
